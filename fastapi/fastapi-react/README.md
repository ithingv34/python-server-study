
### Usage

**Frontend**
```bash
$ cd frontend
$ npm install
$ npm run star
```

**Backend**
- FastAPI v0.78.0
- Python v3.9
- poetry
  - 설치 (ubuntu)
  - `curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python3`


```bash
$ cd backend

# 가상환경 실행
$ poetry shell

# 서버 실행
$ python main.py
```
